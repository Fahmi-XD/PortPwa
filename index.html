About Me

Seorang pelajar yang bersemangat untuk menjelajahi dunia pemrograman, saya memiliki tekad yang kuat untuk menjadi seorang programmer yang handal. Dari usia muda, ketertarikan saya terhadap teknologi dan logika pemrograman telah membimbing langkah-langkah saya. Memecahkan teka-teki dan tantangan membuat saya semakin terpesona dengan dunia coding.

Story

2019

Tahun 2019, saya, sebagai pemula, memasuki dunia pemrograman. Terpesona oleh keajaiban koding, saya memutuskan untuk memulai perjalanan pemrograman saya dengan bahasa Python. Awalnya, sintaks Python yang sederhana membuat saya merasa nyaman menjelajahi dunia logika dan algoritma.

2020

Tahun 2020, semangat belajar saya berkobar. Saya mulai memperdalam pengetahuan saya tentang pengembangan perangkat lunak dan memahami lebih banyak bahasa pemrograman. Proyek-proyek sederhana menjadi latihan sehari-harinya, dan saya menemukan kegembiraan dalam menyelesaikan tantangan pemrograman.

2021

Tahun 2021, fokus saya bergeser ke pengembangan web. Belajar HTML dan CSS membuka pintu menuju dunia desain web, dan saya mulai memahami bagaimana tampilan sebuah halaman web dapat dibentuk. Ilmu saya semakin berkembang seiring pemahaman tentang struktur dan desain tata letak.

2022 - Sekarang

Tahun 2022 hingga sekarang, perjalanan pemrograman membawa saya lebih dalam ke ranah pengembangan web. Saya mulai mendalami peran frontend dan backend, mengeksplorasi kekuatan ReactJS untuk membangun antarmuka pengguna yang dinamis. Pemahaman saya tentang API tumbuh, dan saya terlibat dalam proyek-proyek full-stack, memadukan keterampilan frontend dan backend untuk menciptakan aplikasi web yang lengkap.
